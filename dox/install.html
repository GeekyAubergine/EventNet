Installation Guide

Installation Instructions

This requires the following programs to be installed:

  • PHP
  • MySQL
  • Node.js and npm (Node package manager)

After installing the programs:

  1. Run the command npm install
  2. In the 'inc' folder copy the contents of 'config_sample.php' to 'config.php' and update the variables to use the settings of your environment
  3. In the root folder of the server, run the command sh util/start_eventnet.shThis will start the PHP and Node.js servers

Client instructions

If you are running the server on a server that is not on a server with a domain, you will need to edit the hosts file on all client computers by adding an entry pointing eventnet.com to the IP address of the server.
